## Tools:

- GitLab: open-source solution for managing the full software development lifecycle
- Poetry: tool used for Python packaging and dependency management
- FastAPI: high-performance, easy-to-learn, fast-to-code, ready-for-production web framework for building APIs
- pytest:  makes it easy to write Python tests
- AWS Cloudwatch: monitoring and observability service
- AWS ECR: Amazon Elastic Container Registry
- Terraform: open-source Infrastructure as Code (IaC) software tool
- Codecov: service that helps you track the coverage of your code
- Sentry: service that helps you track unhandled errors inside your applications
